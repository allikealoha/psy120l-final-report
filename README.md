# psy120l-final-report

Experiences in Long-Distance Relationships

This repository contains the published HTML version of my PSY 120L final reproducible report. The report was created using Quarto and R. The published report file is index.html. The dataset is not included in the repository.

This project examined whether perceived communication responsiveness affected relationship satisfaction, and if attachment style and prior experience in long-distance relationships played a role in this association.
